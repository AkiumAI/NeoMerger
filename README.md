## Merge tab

Blend two checkpoints with five merge methods and full per-block control.

<img width="1851" height="919" alt="neomergerui" src="https://github.com/user-attachments/assets/d8a32c32-3642-470c-b1a1-8d1062b10b02" />

**Easy mode** — six semantic sliders map to visual concepts 
instead of block numbers. Five additional experimental sliders 
(Saturation, Contrast, Brightness, Sharpness, Lights & Darks) 
are available in a collapsed section.

<img width="1242" height="566" alt="neomergereasyblocks" src="https://github.com/user-attachments/assets/44525ad0-4510-4094-a748-e7b9057be776" />

**Normal mode** — direct access to all 20 SDXL blocks. 
Toggle descriptions to see what each block influences, with 
warnings for blocks where float values tend to cause issues.

<img width="1217" height="925" alt="neomergerblocksui" src="https://github.com/user-attachments/assets/34632e54-25cb-4288-9428-0557e5a852a6" />

**Block Similarity Analyzer** — before merging, see how different 
the two models are at each block. Color-coded from blue (nearly 
identical) to red (very different), with semantic tags per block.

<img width="686" height="841" alt="neomergeranalyzerui" src="https://github.com/user-attachments/assets/bd06d44a-657e-4e9a-b960-597129e7fa7d" />

## LoRA Tools tab

Bake a LoRA into a checkpoint permanently, or merge two LoRAs 
into one. Handles rank mismatches automatically.

<img width="1834" height="947" alt="neomergerloraui" src="https://github.com/user-attachments/assets/d13e630c-3967-49e1-9bc8-9be0471ad6bf" />

Load any model or LoRA to see its architecture, metadata,
